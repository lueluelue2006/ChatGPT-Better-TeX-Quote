# ChatGPT Better TeX Quote

Userscripts for `https://chatgpt.com/*`.

## Files

- `ChatGPT TeX Copy & Quote 兼容版.js` (MIT)
- `ChatGPT TeX Copy & Quote 整合版.js` (GPL-3.0-or-later)

## What it does

Improves copy/quote behavior on ChatGPT for selections containing KaTeX, preferring original LaTeX (`$...$` / `$$...$$`).
